Summary

This project provides a set of AngularJS directives for the VisJS visualisation library.

Install with bower using

bower install angular-visjs.

Include the js files in your application

angular.js
vis.js
angular-vis.js .

Add 'ngVis' to your module.

Set a directive in your template

<vis-graph2d ng-show="graphLoaded" data="graphData" options="graphOptions" events="graphEvents"></vis-graph2d>

In your code, configure the options.

var dataGroups = new VisDataSet();
dataGroups.add({});
var dataItems = new VisDataSet();
dataItems.add([{}]);
$scope.graphData = {
  items: dataItems,
  groups: dataGroups
};

Option Description
data An object containing two elements items and groups. Each contains a VisDataSet with the items and groups.
options An object containing the vis configuration. Refer to the vis documentation for configuration information.
graph-events An object containing callbacks for vis events. Refer to the vis documentation for events.
Method Description
add Add one or multiple items to the DataSet.
update Update on ore multiple existing items.
getDataSet Returns the dataset.
